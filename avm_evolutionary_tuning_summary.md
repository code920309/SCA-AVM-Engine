# SCA AVM Engine 모델 선정 및 진화적 튜닝 과정 종합 보고서

본 문서는 한국판 부동산 자동가치산정모형(AVM)의 오차율을 최소화하고 모델의 설명력을 극대화하기 위해 수행된 데이터 정제, 외부 API 피처 엔지니어링, 공간 임베딩, 스태킹 앙상블, 그리고 다중 거점 및 OOF 타겟 인코딩에 이르는 전체 기술적 진화 과정을 이모지를 전면 배제하고 기술한 종합 연구 보고서입니다.

---

## 1. 모델 진화 및 최적화 로드맵 요약

SCA AVM Engine은 초기 베이스라인 단계에서부터 최첨단 다중 거점 비선형 피처 모델에 이르기까지 5단계에 걸쳐 정밀 튜닝되었습니다.

```
[1단계: 베이스라인] -> [2단계: 외부 피처] -> [3단계: 공간 임베딩] -> [4단계: 스태킹 앙상블] -> [5단계: v3 다중거점 OOF]
- R²: 0.5626           - R²: 0.5696           - R²: 0.6510           - R²: 0.6738           - R²: 0.7054 (최고 기록)
- MAPE: 41.99%         - MAPE: 36.45%         - MAPE: 31.72%         - MAPE: 30.04%         - MAPE: 30.33% (누수 0.0%)
```

## 2. 각 단계별 아키텍처 설계 및 튜닝 명세

### 1단계: 데이터 정제 및 베이스라인 구축 (Raw Dataset)
* **목적**: 무결결 학습 환경 조성 및 가짜 0값 마스킹.
* **조치 사항**:
  * sggCd가 36110(세종시)인 행 중 sggNm 결측치를 '세종특별자치시'로 복원.
  * floor(층수) 컬럼의 양 끝 공백 제거 후 완전 빈 문자열은 'Unknown' 범주형으로 치환.
  * 집합건물 특성 상 공용면적과 대지지분이 모두 0인 행을 Null 처리 후 중앙값 대체 (가짜 0값 노이즈 드롭).
  * dealAmount 거래 금액 기준 상하위 2% 아웃라이어 드롭.
* **성능 지표**: 결정계수(R²) `0.5626` | 평균 절대 오차율(MAPE) `41.99%`

### 2단계: 외부 행정/입지 API 융합 (Clean Dataset)
* **목적**: 매물의 입지적 요인 및 공법적 제한 사항 물리 피처 결합.
* **조치 사항**:
  * 세움터 표제부 연계를 통한 건폐율(bcRat) 및 용적률(vlRat) 매핑 및 결측치는 읍면동 평균값 대체.
  * 카카오 로컬 API를 연동하여 최인접 지하철역 거리(dist_to_subway), 경유 노선 수(subway_line_count), 반경 200m 내 카페 수(cafe_count_200m) 결합.
* **성능 지표**: 결정계수(R²) `0.5696` | 평균 절대 오차율(MAPE) `36.45%`

### 3단계: 시공간 융합 임베딩 (Spatiotemporal Embedding)
* **목적**: 위경도의 지리적 상관관계 및 가격 모멘텀 비선형성 학습.
* **조치 사항**:
  * **KNN 공간 표면 (spatial_knn_price)**: Harversine 거리를 기반으로 타겟 단가를 예측한 전국구 공간 보정 표면 구축 (n_neighbors=15, weights='distance').
  * **트리 공간 표면 (spatial_tree_price)**: 5-Fold OOF 구조 하에서 HistGradientBoostingRegressor를 구동하여 위경도로만 타겟 단가를 학습한 계단식 영역 경계면 형성.
  * **하이브리드 모델**: 두 공간 피처를 동시에 이식하여 연속 거리 곡면과 지리적 단절선을 동시에 학습.
* **성능 지표**: 결정계수(R²) `0.6510` | 평균 절대 오차율(MAPE) `31.72%`

### 4단계: 고차원 Stacking 앙상블 모형
* **목적**: 범주형 임베딩 강점인 CatBoost와 연속형 부스팅 강점인 HGBR의 장점 융합.
* **조치 사항**:
  * **Level 0 베이스 모델**:
    * HistGradientBoostingRegressor (max_iter=300, max_depth=10)
    * CatBoostRegressor (iterations=500, depth=8, 원본 텍스트 카테고리 임베딩 가동)
  * **Level 1 메타 모델**: Ridge(alpha=1.0)를 지정하여 5-Fold Stacking CV 내부 격리 구조 하에서 과적합 없이 가중치 융합.
* **성능 지표**: 결정계수(R²) `0.6738` | 평균 절대 오차율(MAPE) `30.04%`

### 5단계: v3 다중 거점 및 Leakage-Free OOF 타겟 인코딩 (최종 최적 모델)
* **목적**: 데이터 누수의 완벽 차단 및 극세사 해상도 미시 입지 가중치 확보.
* **조치 사항**:
  * **Leakage-Free OOF 읍면동 인코딩 (umd_encoded_oof)**: 기존의 전체 평균 매핑으로 인한 타겟 누수를 차단하기 위해, 5-Fold 교차 검증 내부 루프에서 오직 훈련 폴드 데이터로만 읍면동별 평균 단가를 계산하여 검증 폴드에 매핑하는 엄격한 격리 장치 구축.
  * **다중 상권 거점 피처 (dist_to_closest_hotspot)**: 각 시군구(sggNm) 내 실거래가 상위 5% 매물 좌표를 '미시 상권 핵심지(Hotspot)'로 지정하고 최인접 최단 하버사인 거리(km) 연산.
  * **모델 하이퍼 가중치 극대화**: max_iter=700, learning_rate=0.025, l2_regularization=5.0으로 규제를 대폭 강화하여 복잡한 비선형 입지 지표들의 과적합 방어.
* **성능 지표**: 결정계수(R²) `0.7054` | 평균 절대 오차율(MAPE) `30.33%`

## 3. 손실 함수 설계에 관한 수학적 실증 결과 (MSE vs MAE)

오차율(MAPE) 최소화를 목표로 MAE 손실 함수와 샘플 가중치를 직접 주입한 최적화 실험을 추가 수행하여 다음과 같은 분석 결론을 확보하였습니다.

### 실험 결과 대조
1. **로그 제곱 오차 (squared_error)**: 결정계수 `0.7054` | MAPE `30.33%`
2. **로그 절대 오차 (absolute_error) + 단가 역비례 가중치**: 결정계수 `0.6880` | MAPE `31.27%`

### 수학적 고찰
* **왜도 완화와 기하 평균**: 타겟 단가가 로그 변환(`np.log1p`)된 상태에서 제곱 오차(MSE)를 사용하면 모델은 로그 공간의 평균을 예측하도록 최적화됩니다. 이는 원래 스케일에서 이상치의 기하학적 영향을 자연스럽게 제어하는 기하 평균(Geometric Mean)에 해당하며, 궁극적으로 비율 오차인 MAPE의 정의적 최소화 노선과 부합하게 됩니다.
* **부스팅 트리의 안정성**: 개별 샘플의 단가 역비례 가중치를 서로 다르게 주입할 시, 그래디언트 부스팅 트리가 최적 리프 분할을 찾는 과정에서 Gradient와 Hessian의 합산 균등성이 훼손되어 약간의 모델 분산(Variance) 증가를 초래했습니다.
* **결론**: 로그 타겟 변환 조건 하에서는 `squared_error`(MSE) 손실함수가 가장 완벽한 일반화 성능을 띱니다.
